# Customer-Behavior-Analysis
This project analyzes customer purchasing behavior and visualizes key insights using Power BI. The goal of this project is to understand customer trends, subscription status, and revenue distribution across product categories.
The dataset was first cleaned and processed using Python (Pandas) to remove inconsistencies and prepare it for analysis. Then SQL queries in PostgreSQL (pgAdmin4) were used to extract and organize the required data. Finally, an interactive dashboard was created in Power BI to present the insights visually.

Tools & Technologies Used
Python (Pandas) – Data Cleaning & Preprocessing
PostgreSQL – Data Querying
pgAdmin4 – Database Management
Power BI – Data Visualization & Dashboard Creation

Features of the Dashboard
KPI Cards showing:
Total Number of Customers
Average Purchase Amount
Average Customer Review Rating
Customer Subscription Analysis
Percentage of subscribed vs non-subscribed customers
Revenue Analysis
Category-wise revenue distribution (Clothing, Accessories, Footwear, Outerwear)

Interactive Filters
Allows users to explore customer behavior dynamically
Workflow
Raw customer dataset collected.
Data cleaning performed using Python Pandas.
Data stored and queried using PostgreSQL in pgAdmin4.
Processed data imported into Power BI.
Interactive dashboard created for business insights.

Key Insights
Majority of customers are non-subscribers.
Clothing category generates the highest revenue.
Average purchase amount and customer ratings help evaluate overall customer satisfaction.

Purpose of the Project
This project demonstrates data cleaning, SQL querying, and data visualization skills required for data analytics and business intelligence roles.
